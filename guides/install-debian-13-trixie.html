Install Debian 13 (Trixie)

Debian 13 "Trixie" is the current stable release, released August 9, 2025. Current point release is 13.5 (May 16, 2026). Supported until August 2030 (including LTS).

Debian 12 (Bookworm) reached end of its full support period in June 2026 and is in LTS until 2028. For new installations, use Debian 13.

Debian 13 ships with Linux kernel 6.12 LTS, GNOME 48, KDE Plasma 6.3, OpenSSL 3.5, and OpenJDK 21.

Download

Go to debian.org/distrib/netinst and download:

  • amd64 netinst — small, downloads packages during install (needs internet)
  • DVD image — larger, installs without internet

If your Wi-Fi card needs proprietary firmware — most modern laptops do — download the unofficial non-free firmware image from cdimage.debian.org. This is the version most people should use.

Installer Type

Graphical install: Easier, same result. Use for desktop systems.

Expert install: More control — useful for servers, custom partitioning, and specific locale setups.

Installation

Hostname: Set something meaningful — e.g. debian-server or homelab.

Root password: You have two options:

  • Set a root password — root is enabled, your user needs explicit sudo setup
  • Leave blank — root is disabled, your user automatically gets sudo (Ubuntu-style)

Partitioning: For most setups, Guided — use entire disk with LVM is the right choice.

Software selection (tasksel):

  • Desktop: check your preferred desktop environment (GNOME, KDE, XFCE, etc.) + standard system utilities
  • Server: check SSH server + standard system utilities only (no desktop)

Confirm and install. Reboot when done.

After Install — Enable sudo (If You Set a Root Password)

If you set a root password during install, your user was not automatically added to sudo:

su -
apt install sudo -y
usermod -aG sudo yourusername
exit

Log out and back in. Test: sudo apt update

Enable Non-Free Repositories

sudo nano /etc/apt/sources.list

Add contrib non-free non-free-firmware to each line:

deb http://deb.debian.org/debian/ trixie main contrib non-free non-free-firmware
deb http://security.debian.org/debian-security trixie-security main contrib non-free non-free-firmware
deb http://deb.debian.org/debian/ trixie-updates main contrib non-free non-free-firmware
sudo apt update

Install Common Desktop Apps

# Flatpak (gives access to more up-to-date apps than Debian packages)
sudo apt install flatpak -y
flatpak remote-add --if-not-exists flathub https://dl.flathub.org/repo/flathub.flatpakrepo

# Install apps via Flatpak
flatpak install flathub com.visualstudio.code
flatpak install flathub org.videolan.VLC
flatpak install flathub com.spotify.Client

Upgrading from Debian 12 to Debian 13

sudo apt update && sudo apt full-upgrade -y
sudo nano /etc/apt/sources.list
# Change all instances of "bookworm" to "trixie"
sudo apt update
sudo apt full-upgrade -y
sudo reboot

Review the official Debian 13 release notes for upgrade-specific warnings before proceeding. In particular: there is no direct upgrade path from Debian 11 — you must go through 12 first.
