# macOS Accessibility Features — Hidden Productivity Tools

macOS accessibility features are useful for everyone, not just those with disabilities. Many are the fastest way to do common tasks.

---

## Zoom

**System Settings → Accessibility → Zoom**

- Use keyboard shortcut: Option+Command+= to zoom in, Option+Command+- to zoom out
- **Use scroll gesture with modifier keys** — hold Control and scroll to zoom the entire screen
- Hover text (Cmd+hover) shows large text tooltip for whatever the cursor is over

---

## Dictation

**System Settings → Keyboard → Dictation → On**

Press **Fn** twice (or your custom shortcut) to start dictating. Works in any text field. Uses on-device processing — nothing sent to Apple.

```
"Click" — clicks
"Select all" — selects all text
"New paragraph" — inserts a paragraph break
"Delete that" — deletes the last word
```

---

## Voice Control

**System Settings → Accessibility → Voice Control**

More powerful than Dictation — control the entire Mac by voice. Say "Show numbers" to see clickable numbers appear on every interactive element. Say "Click 3" to click element 3. No keyboard or mouse needed.

---

## Display Accommodations

**System Settings → Accessibility → Display:**

- **Reduce Motion** — stops animations (also speeds up the UI noticeably)
- **Reduce Transparency** — makes sidebar and menu bar opaque (better readability)
- **Increase Contrast** — darker borders on all UI elements
- **Colour Filters** — protanopia, deuteranopia, tritanopia, greyscale

---

## Pointer Control

**System Settings → Accessibility → Pointer Control:**

- Increase cursor size — easier to find on large monitors
- Shake cursor to locate — shake mouse and the cursor grows temporarily
- Ignore built-in trackpad when mouse is connected

---

## Keyboard Shortcuts for Accessibility

| Shortcut | Action |
|---|---|
| **Cmd+Option+F5** | Open Accessibility Shortcuts panel |
| **Ctrl+Scroll** | Zoom in/out |
| **Cmd+Option+=** | Zoom in (with zoom enabled) |
| **Fn Fn** | Start Dictation |
| **Cmd+Option+8** | Toggle Classic Invert colours |

---

## Text to Speech

**System Settings → Accessibility → Spoken Content:**

- **Speak Selection** — press a keyboard shortcut to read selected text aloud
- **Speak Screen** — two-finger swipe down from top reads the whole screen
- **Typing Feedback** — speak characters and words as you type

---

## Switch Control

Control Mac from any single button — keyboard key, switch, or external device. Useful for accessibility but also for automation — set up a foot pedal to trigger actions.
