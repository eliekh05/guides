Take Screenshots on Linux

How you take screenshots on Linux depends on two things: your desktop environment and whether you are running X11 or Wayland. Check which you are on:

echo $XDG_SESSION_TYPE
# Returns: x11 or wayland

GNOME (Ubuntu, Fedora, Pop!_OS)

GNOME has a built-in screenshot tool that works on both X11 and Wayland.

ShortcutWhat it captures
Print ScreenOpens screenshot UI — choose full screen, window, or selection
Alt + Print ScreenActive window only (GNOME 42+)
Shift + Print ScreenSelect a region
Ctrl + any of the aboveCopy to clipboard instead of saving

Screenshots save to ~/Pictures/Screenshots/.

KDE Plasma — Spectacle

KDE ships with Spectacle — a full-featured screenshot tool that works on both X11 and Wayland.

ShortcutAction
Print ScreenFull screen capture
Meta + Print ScreenActive window
Shift + Print ScreenRectangular region selection

Spectacle supports annotation, delay, and copy to clipboard. Screenshots save to ~/Pictures/ by default — change in Spectacle settings.

Hyprland — grim + slurp

Hyprland is a Wayland compositor so X11 tools like scrot do not work. Use grim (capture) and slurp (region selection):

# Install
sudo pacman -S grim slurp wl-clipboard   # Arch
sudo apt install grim slurp wl-clipboard  # Ubuntu/Debian

# Full screen
grim screenshot.png

# Select a region interactively
grim -g "$(slurp)" screenshot.png

# Copy selection directly to clipboard
grim -g "$(slurp)" - | wl-copy

For Hyprland specifically, grimblast (the official Hyprland screenshot utility) wraps grim and slurp with extra features:

# Install via AUR
yay -S grimblast-git

# Capture active window
grimblast copy active

# Select region and copy to clipboard
grimblast copy area

# Save to file
grimblast save area screenshot.png

Add keybindings to your Hyprland config (~/.config/hypr/hyprland.conf):

bind = , Print, exec, grimblast copy area
bind = SHIFT, Print, exec, grimblast copy active
bind = SUPER, Print, exec, grimblast copy output

Flameshot — Annotation and Markup

Flameshot adds annotation tools — arrows, text, blur, highlight. Best for sharing screenshots with markups.

# Install
sudo apt install flameshot -y       # Ubuntu/Debian
sudo dnf install flameshot -y       # Fedora
sudo pacman -S flameshot            # Arch

# Launch interactive UI
flameshot gui

On GNOME Wayland, if flameshot shows a blank screen:

env XDG_CURRENT_DESKTOP=sway flameshot gui

On Hyprland, flameshot needs xdg-desktop-portal-hyprland installed and running. If it does not work, use grimblast instead.

Command Line — X11 (scrot)

scrot works on X11 sessions only. If you are on Wayland, use grim instead.

# Install
sudo apt install scrot -y

# Full screen saved to current folder
scrot screenshot.png

# Select a region interactively
scrot -s screenshot.png

# 5-second delay then capture
scrot -d 5 screenshot.png

# Active window only
scrot -u screenshot.png

# Capture and open in viewer
scrot screenshot.png && eog screenshot.png

Command Line — Wayland (grim)

# Full screen
grim screenshot.png

# Specific monitor (get monitor names with: wayland-info | grep name)
grim -o eDP-1 screenshot.png

# Region selection
grim -g "$(slurp)" screenshot.png

# Copy to clipboard
grim - | wl-copy
