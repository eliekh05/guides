Windows Keyboard Shortcuts That Actually Save Time

The ones worth memorizing — not every shortcut, just the ones you will use every day.

Window Management

Shortcut What it does
Win + D Show/hide desktop
Win + E Open File Explorer
Win + L Lock screen
Win + ← Snap window to left half
Win + → Snap window to right half
Win + ↑ Maximise window
Win + ↓ Restore / minimise window
Win + Shift + S Screenshot — select area (saves to clipboard)
Alt + F4 Close current window
Win + Tab Task view — see all open windows and virtual desktops
Ctrl + Win + D Create a new virtual desktop
Ctrl + Win + ←/→ Switch between virtual desktops

System and Utilities

Shortcut What it does
Win + X Power user menu (Device Manager, Terminal, etc.)
Win + I Open Settings
Win + A Open Quick Settings (Wi-Fi, Bluetooth, volume)
Win + N Open Notification Centre
Win + K Cast / connect to display or Bluetooth
Ctrl + Shift + Esc Open Task Manager directly
Win + R Run dialog — type commands like cmd, msconfig, notepad
Win + . Emoji picker
Win + V Clipboard history (must enable first in Settings)
Win + P Project display — duplicate, extend, second screen only

File Explorer

Shortcut What it does
Alt + ← Go back
Alt + → Go forward
Alt + ↑ Go up one folder level
Ctrl + N Open new File Explorer window
Ctrl + W Close current window
F2 Rename selected file
F5 Refresh
Ctrl + Shift + N Create new folder
Alt + Enter Open Properties for selected file
Ctrl + L Select the address bar

Text Editing

Shortcut What it does
Ctrl + Z Undo
Ctrl + Y Redo
Ctrl + A Select all
Ctrl + C / X / V Copy / Cut / Paste
Ctrl + Home / End Jump to beginning / end of document
Ctrl + ← / → Jump word by word
Shift + ← / → Select one character at a time
Ctrl + Shift + ← / → Select one word at a time

Browser (Chrome, Edge, Firefox)

Shortcut What it does
Ctrl + T New tab
Ctrl + W Close tab
Ctrl + Shift + T Reopen last closed tab
Ctrl + L Select address bar
Ctrl + Tab Next tab
Ctrl + Shift + Tab Previous tab
F12 Open Developer Tools
Ctrl + Shift + Delete Clear browsing data
Ctrl + +/- Zoom in/out
Ctrl + 0 Reset zoom

Run Dialog Shortcuts (Win + R)

Type these in the Run dialog for instant access:

Type this Opens
cmd Command Prompt
powershell PowerShell
ms-settings: Settings
msconfig System Configuration
devmgmt.msc Device Manager
diskmgmt.msc Disk Management
taskschd.msc Task Scheduler
regedit Registry Editor
control Control Panel
appwiz.cpl Add/Remove Programs
%temp% Temp folder
%appdata% AppData/Roaming folder
shell:startup Startup folder
