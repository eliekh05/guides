Clean Install Arch Linux + Hyprland (Wi-Fi Only)

This guide installs Arch Linux using the official archinstall script, then sets up Hyprland using the JaKooLit automated installer. Aimed at Wi-Fi only machines.

Always download the latest Arch Linux ISO from archlinux.org/download. Do not use pinned old ISO versions — Arch is a rolling release and the installer downloads packages live from the internet anyway.

Requirements

  • A USB drive — 2 GB minimum
  • A machine with UEFI (most hardware from 2012 onwards)
  • At least 20 GB free disk space, 4 GB RAM recommended
  • Wi-Fi network you know the password to

Create the Bootable USB

Download the latest ISO from archlinux.org/download.

Flash it using Ventoy (recommended — lets you boot multiple ISOs from one USB) or Rufus on Windows or balenaEtcher on any platform.

Boot from USB

  1. Insert USB and restart.
  2. Enter your boot menu — usually F12, F9, F2, or DEL depending on manufacturer.
  3. Select the USB drive.
  4. At the GRUB menu choose the first option.

You land at a root shell. The live environment is minimal by design.

Connect to Wi-Fi

Use iwctl — the interactive Wi-Fi tool included in the live ISO:

iwctl

Inside iwctl:

# Find your wireless interface name (usually wlan0)
device list

# Scan for networks
station wlan0 scan

# Show available networks
station wlan0 get-networks

# Connect (replace SSID with your network name)
station wlan0 connect "Your Network Name"

# For hidden networks
station wlan0 connect-hidden "Your Network Name"

exit

Verify you have an IP and internet:

ip a
ping -c 3 archlinux.org

If ping works, you are ready.

Run archinstall

archinstall

Navigate with arrow keys, Enter to select, Escape to go back. Work through each section:

Mirrors

A list of countries appears. Press Space to select each country — select all of them. When the list is done, confirm. Reflector runs automatically, tests all the mirrors from your selected countries by speed, and picks the fastest ones. You do not need to choose any specific mirror yourself.

Disk Configuration

Choose Use a best-effort default partition layout unless you know what you are doing. Select your target disk. Choose ext4 as the filesystem unless you want btrfs (which supports snapshots — useful but adds complexity).

If you want disk encryption, enable it here. You will need to enter a passphrase every boot.

Profile

Select None — do not install a desktop environment here. JaKooLit's script installs Hyprland and everything it needs. Installing a DE here creates conflicts.

Audio

Select Pipewire. PulseAudio is legacy — everything in 2026 expects Pipewire.

Network Configuration

Select NetworkManager. This gives you GUI Wi-Fi management after install and works with most tools.

Other Settings

  • Hostname: anything you want
  • Root password: set one
  • User account: create one, check the sudo box
  • Timezone: your timezone
  • NTP: Yes (keep automatic time sync on)
  • Additional packages: add git — you need it for the Hyprland installer

Install

Select Install at the bottom. Confirm and wait — 5 to 20 minutes depending on your internet and mirror selection.

When asked to chroot after install, type no and reboot. Remove the USB when the machine restarts.

First Boot — Log In as Your User

You land at a terminal login. Log in with the user you created. You should have internet automatically via NetworkManager if you are on Ethernet. For Wi-Fi:

nmcli device wifi connect "Your Network Name" password "yourpassword"

Install Hyprland with JaKooLit

The recommended method is cloning the repo directly — more reliable than piping curl to sh:

git clone --depth=1 https://github.com/JaKooLit/Arch-Hyprland.git ~/Arch-Hyprland
cd ~/Arch-Hyprland
chmod +x install.sh
./install.sh

The script asks a series of yes/no questions. Here is what each option does so you can make your own choice:

OptionWhat it doesRecommend
gtk_themesGTK dark/light theme supportYes
bluetoothBluetooth stack + blueman GUIYes if you use Bluetooth
thunarFile manager with pluginsYes
quickshellDesktop overview widgetYes
sddmDisplay manager (login screen)Yes

pokemonPokémon color display in terminalYour choice (cosmetic only)

dotsJaKooLit's preconfigured dotfilesYes — this is the whole point
input_groupAdds your user to the input groupYes

Do not enable both nvidia and nouveau. They conflict. If you have Nvidia hardware, enable nvidia and leave nouveau off. If you have AMD or Intel, leave both off.

When asked about the 1 GB wallpaper pack — choose based on your storage. The rainbow wallpaper set is smaller and fine to say yes to.

The installation takes 10–30 minutes. When complete, reboot.

After Reboot

SDDM presents a login screen. Log in. Hyprland starts.

Default keybindings (Super = Windows/Command key):

  • Super + Return — open terminal
  • Super + E — file manager
  • Super + R or Super + Space — app launcher
  • Super + Q — close window
  • Super + 1–9 — switch workspace
  • Super + Shift + 1–9 — move window to workspace

Common Issues

Black screen after login: Usually a GPU driver issue. TTY in with Ctrl+Alt+F2 and check journalctl -xe | grep hyprland.

No sound: Run wpctl status to check Pipewire. Try systemctl --user restart pipewire.

Screen sharing not working in Discord/OBS: You skipped xdph during install. Re-run the installer or manually install xdg-desktop-portal-hyprland.

Wi-Fi dropped after reboot: nmcli device wifi connect "Name" password "pass" to reconnect. Should auto-connect on subsequent boots.
