# Enable God Mode in Windows

God Mode is a hidden folder that puts every Windows settings and control panel option in one place — over 200 settings, tools, and shortcuts in a single window. It works on Windows 10 and 11.

---

## Enable It

1. Right-click your **Desktop** → **New** → **Folder**.
2. Name the folder exactly this (copy and paste):

```
GodMode.{ED7BA470-8E54-465E-825C-99712043E01C}
```

3. Press **Enter**.

The folder icon will change to a Control Panel icon. Double-click it to open.

---

## What Is Inside

Every Windows settings shortcut organised into categories including:

- Administrative Tools
- Backup and Restore
- BitLocker settings
- Color Management
- Credential Manager
- Date and Time
- Default Programs
- Device Manager
- Display settings
- Ease of Access
- File Explorer options (show/hide hidden files, extensions, etc.)
- Folder Options
- Fonts
- HomeGroup
- Indexing Options
- Internet Options
- Keyboard settings
- Mouse settings
- Network and Sharing Centre
- Pen and Touch
- Phone and Modem
- Power Options
- Programs and Features
- Recovery
- Region
- Security and Maintenance
- Sound
- Speech Recognition
- Storage Spaces
- Sync Centre
- System
- Taskbar and Navigation
- Troubleshooting
- User Accounts
- Windows Defender Firewall
- Windows Update

## Other Hidden Folders

The same trick works with other GUIDs for more specific panels:

| Folder Name | What it Opens |
|---|---|
| `FontsFolder.{BD84B380-8CA2-1069-AB1D-08000948F534}` | Fonts folder |
| `NetworkConnections.{7007ACC7-3202-11D1-AAD2-00805FC1270E}` | Network Connections |
| `PowerOptions.{025A5937-A6BE-4686-A844-36FE4BEC8B6D}` | Power Options |

Create a new folder and paste any of these as the name — the same trick works.

---

> **To remove:** right-click the God Mode folder → **Delete**. It just deletes the shortcut folder, nothing else.
