Take Screenshots on macOS

macOS has built-in screenshot shortcuts that work everywhere — no extra apps needed.

Keyboard Shortcuts

ShortcutWhat it captures
Cmd + Shift + 3Entire screen
Cmd + Shift + 4Drag to select an area
Cmd + Shift + 4, then SpaceClick any window to capture just that window
Cmd + Shift + 5Opens screenshot toolbar — all options plus screen recording
Cmd + Shift + 6Touch Bar only (older MacBook Pro)

Add Ctrl to any shortcut to copy to clipboard instead of saving as a file. For example: Cmd + Ctrl + Shift + 4 copies a selection to clipboard.

Where Screenshots Are Saved

By default screenshots save to your Desktop as .png files named Screenshot YYYY-MM-DD at HH.MM.SS.png.

Change the save location

Press Cmd + Shift + 5 → click Options → choose a folder under Save to. You can pick Desktop, Documents, Clipboard, or any other folder.

Screenshot Toolbar (Cmd + Shift + 5)

The toolbar at the bottom of the screen lets you:

  • Capture entire screen, a window, or a selection
  • Record the entire screen or a selection as a video
  • Set a timer (5 or 10 seconds) before capture
  • Show or hide the cursor in screenshots

Annotate After Taking a Screenshot

When you take a screenshot, a thumbnail appears in the bottom-right corner. Click it before it disappears to open Markup — draw, add text, crop, and sign directly on the screenshot before saving.

Screenshot via Terminal

# Capture full screen to a file
screencapture screenshot.png

# Interactive selection mode
screencapture -i screenshot.png

# Copy to clipboard instead of saving
screencapture -c

# 5-second delay
screencapture -T 5 screenshot.png

# Capture a specific window by window ID
screencapture -l WINDOW_ID screenshot.png
