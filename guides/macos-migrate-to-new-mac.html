Migrate to a New Mac — Complete Guide

Migration Assistant transfers your apps, files, settings, and accounts from your old Mac to your new one.

Before You Start

On your old Mac:

  1. Install all pending macOS updates
  2. Sign out of apps that limit device counts: Adobe Creative Cloud, Microsoft Office, some DRM software
  3. Make sure both Macs are on the same WiFi network (or use a cable)
  4. Charge both Macs or plug them in

Method 1 — Migration Assistant over WiFi (Easiest)

On your new Mac (during setup wizard or after):

  1. Open Migration Assistant (Applications → Utilities)
  2. Select From a Mac, Time Machine backup, or startup disk
  3. Click Continue

On your old Mac:

  1. Open Migration Assistant (same app)
  2. Select To another Mac
  3. Click Continue

Both Macs will show a code — verify they match, then click Continue on both.

Choose what to transfer:

  • Applications — all installed apps
  • User Account — your documents, desktop, downloads, settings
  • System and Network — network settings

Click Continue. Transfer time varies — a few minutes for small amounts, several hours for 500GB+.

Method 2 — Migration Assistant from Time Machine Backup

  1. Open Migration Assistant on the new Mac
  2. Select From a Mac, Time Machine backup, or startup disk
  3. Connect your Time Machine drive
  4. Select the backup and most recent snapshot

Method 3 — Direct Cable Connection

For faster transfers (especially large amounts of data):

Intel Macs: use a Thunderbolt cable Apple Silicon to Apple Silicon: use a USB-C cable

Connect the cable, then open Migration Assistant on both Macs.

What Gets Transferred

✅ Transferred:

  • Applications
  • User files (Documents, Downloads, Desktop, Pictures, etc.)
  • User settings and preferences
  • Email accounts and messages (Mail app)
  • Safari bookmarks and history
  • Contacts, Calendar, Notes
  • WiFi networks and passwords
  • Printers
  • Screen Saver and Desktop settings

⚠️ May need re-activation:

  • Microsoft Office (sign in again)
  • Adobe Creative Cloud (sign in again)
  • Licensed apps with device limits
  • App Store apps (usually just sign in with Apple ID)

❌ Not transferred:

  • FileVault state (encryption must be re-enabled on new Mac)
  • Some app data stored in non-standard locations

After Migration

  1. Check System Settings → Apple ID — sign in if not already
  2. App Store → Updates — check for updates to transferred apps
  3. Re-activate any licensed software
  4. System Settings → Printers & Scanners — verify printers
  5. Test your important apps before wiping the old Mac
