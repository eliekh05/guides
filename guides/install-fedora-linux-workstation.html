Install Fedora Linux Workstation

Fedora ships cutting-edge software faster than any other major distro. The current release is Fedora 44 (released April 2026) with GNOME 50. Each release is supported for about 13 months.

Fedora 42 reached end of life in April/May 2026. If you are running 42 or 43, upgrade now: sudo dnf system-upgrade download --releasever=44 && sudo dnf system-upgrade reboot

Download

Go to fedoraproject.org/workstation and download the ISO. Flash with Ventoy or balenaEtcher.

Fedora also offers official spins — if you prefer KDE Plasma (now a full Edition since Fedora 42), download from fedoraproject.org/spins/kde.

Install

Boot from USB. Fedora 42+ uses a new web-based installer by default on Workstation.

Language and keyboard: Select yours.

Installation destination: Click your disk. Choose Automatic for simple setup. For dual boot, choose Custom and create a free space partition. The new guided partitioning in Fedora 42+ makes this significantly easier than before — select your goal (install alongside existing OS, use entire disk, etc.) and it handles the rest.

User creation: Create a user and check Make this user administrator.

Click Begin Installation and wait 10–20 minutes.

First Steps After Install

Update everything first

sudo dnf upgrade -y

Or: Software → Updates → Install All

Enable RPM Fusion

Fedora excludes proprietary and patent-encumbered software by default. RPM Fusion adds Nvidia drivers, media codecs, and more:

sudo dnf install \
  https://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm \
  https://download1.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-$(rpm -E %fedora).noarch.rpm

Install media codecs

sudo dnf groupupdate multimedia --setopt="install_weak_deps=False" --exclude=PackageKit-gstreamer-plugin
sudo dnf groupupdate sound-and-video

Fedora ships ffmpeg-free by default which has known bugs. Switch to the RPM Fusion version for better reliability:

sudo dnf swap ffmpeg-free ffmpeg --allowerasing

Enable Flathub

flatpak remote-add --if-not-exists flathub https://dl.flathub.org/repo/flathub.flatpakrepo

Install common apps

# Via dnf
sudo dnf install vim git curl wget htop gnome-tweaks

# Via Flatpak (more up-to-date)
flatpak install flathub com.spotify.Client
flatpak install flathub com.discordapp.Discord
flatpak install flathub org.videolan.VLC
flatpak install flathub com.visualstudio.code

Nvidia drivers (if you have Nvidia GPU)

Install after enabling RPM Fusion:

sudo dnf install akmod-nvidia xorg-x11-drv-nvidia-cuda
sudo reboot

After reboot, verify:

nvidia-smi

Firmware updates

Fedora integrates with LVFS for BIOS and hardware firmware updates:

sudo fwupdmgr refresh
sudo fwupdmgr get-updates
sudo fwupdmgr update

Enable SSD TRIM

sudo systemctl enable fstrim.timer

Upgrading to a New Fedora Version

sudo dnf upgrade --refresh
sudo dnf install dnf-plugin-system-upgrade
sudo dnf system-upgrade download --releasever=44
sudo dnf system-upgrade reboot

Back up before upgrading. The upgrade reboots into a special offline mode and takes 20–40 minutes.
